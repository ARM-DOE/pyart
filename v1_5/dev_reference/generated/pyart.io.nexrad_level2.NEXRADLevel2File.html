pyart.io.nexrad_level2.NEXRADLevel2File¶

class pyart.io.nexrad_level2.NEXRADLevel2File(filename)[source]¶

Class for accessing data in a NEXRAD (WSR-88D) Level II file.

NEXRAD Level II files [R15], also know as NEXRAD Archive Level II or WSR-88D Archive level 2, are available from the NOAA National Climate Data Center [R16] as well as on the UCAR THREDDS Data Server [R17]. Files with uncompressed messages and compressed messages are supported. Currently only “message 31” type files are supported, “message 1” file cannot be read using this class.

Parameters:

filename : str

Filename of Archive II file to read.

References

[R15](1, 2) http://www.roc.noaa.gov/WSR88D/Level_II/Level2Info.aspx
[R16](1, 2) http://www.ncdc.noaa.gov/
[R17](1, 2) http://thredds.ucar.edu/thredds/catalog.html

Attributes

nscans (int) Number of scans in the file.
scan_msgs (list of arrays) Each element specifies the indices of the message in msg31s which belong to a given scan.
volume_header (dict) Volume header.
vcp (dict) VCP information dictionary.
_records (list) A list of all records (message) in the file.
_fh (file-like) File like object from which data is read.
