pyart.io.nexrad_cdm._gen_vnames¶

pyart.io.nexrad_cdm._gen_vnames(field, high_res)[source]¶

Return a dictionary of variables/dimension names in a CDM dataset.

Parameters :

field : ‘refl’ or ‘doppler’

Type of fields to generate dictionary for.

high_res: bool :

True if the fields are high resolution.
Returns :

vnames : dic

Dictionary of variable and dimension names in the thredds server dataset for a given field type. For use as the vnames parameter in _radar_from_cdm.
