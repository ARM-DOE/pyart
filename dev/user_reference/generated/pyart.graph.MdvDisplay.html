pyart.graph.MdvDisplay¶

class pyart.graph.MdvDisplay(mdvfile)[source]¶

A display object for creating plots from data in a MdvFile objects.

Parameters :

mdvfile : MdvFile

MdvFile object to use for creating plots.

Attributes

plots (list) List of plots created.
plot_vars (list) List of fields plotted, order matches plot list.
cbs (list) List of colorbars created.
loc ((float, float)) Latitude and Longitude of radar in degrees.
origin (‘Radar’) ‘Radar’
time_begin (datetime) Time of first radar scan.
radar_name (str) Radar name.
mdvfile (MdvFile) MdvFile object use to create plots.
