Input and output (pyart.io)Â¶

Py-ART has modules, classes and functions which are able to read data from and write data to a number of file formats.

read_rsl(filename[, radar_format, callid, ...]) Read a file supported by RSL
read_mdv(filename) Read a MDV file.
read_netcdf(filename) Read a netCDF file.
write_netcdf(filename, radar[, format]) Write a Radar object to a CF/Radial compliant netCDF file.
read_grid(filename) Read a netCDF grid file
write_grid(filename, pygrid[, format]) Write a PyGrid object to a CF-1.5 and ARM standard netcdf file
Radar(time, _range, fields, metadata, ...[, ...]) A class for storing antenna coordinate radar data.
PyGrid(fields, axes, metadata) An object for holding gridded Radar data.
